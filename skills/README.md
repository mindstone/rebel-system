# Skills

Efficient, composable patterns and best practices for common operations by AI agents.


## Skill categories

- `Anthropic-official-skills/` — Reference implementations from Anthropic's open-source skills repository
- `coding/` — Git operations, Python setup, product specs, library selection
- `communication/` — Slack and Gmail MCP workflows
- `demo/` — Example skills for demonstration purposes
- `documentation/` — Writing skills, planning docs, deep-dives, diagrams, editing transcripts
- `external-IDE-OBSOLETE/` — Cursor and other IDE integrations
- `legal/` — Contract analysis, risk assessment, plain-language legal document summaries
- `meetings/` — Calendar availability, meeting prep, follow-ups, weekly briefs, meeting dynamics analysis
- `memory/` — Memory setup, population, and updates
- `operations/` — Use case discovery, hiring processes, wins and learnings
- `research/` — Web research, internal CRM research, Notion/Slack search, knowledge extraction, content transformation
- `safety/` — Memory sensitivity and safety guard skills
- `sales/` — Sales proposal drafting
- `system/` — Workspace setup, file organization, MCP management, space management
- `thinking/` — Sounding board mode, prioritization, devil's advocate, complex task planning, problem-solving frameworks (inversion, collision-zone, scale-game, simplification-cascades, meta-patterns, productive tensions)
- `utilities/` — Calendar events, Excel/PDF extraction, string manipulation, export tools
