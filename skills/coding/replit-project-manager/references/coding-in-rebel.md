# Coding in Rebel

If you want **Rebel** to write the code instead of delegating to Replit Agent, use the [software-engineer workflow](rebel-system/skills/workflows/software-engineer/SKILL.md).

**When to use Replit Agent:**
- Building a new app from scratch (0-to-1)
- You want a hosted, deployed result without managing infrastructure
- The project is self-contained on Replit

**When to use Rebel's software-engineer workflow:**
- Working on an existing codebase (local or Git repo)
- You need precise, surgical edits across many files
- The project is complex with established architecture
- You want fine-grained control over implementation decisions
