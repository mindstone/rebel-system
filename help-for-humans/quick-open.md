# Quick Open

Press **Cmd+O** (Mac) or **Ctrl+O** (Windows/Linux) to instantly open any file in your Library.

## How It Works

1. **Press the shortcut** — A search dialog appears
2. **Start typing** — Filter by filename, skill name, or memory topic
3. **Use arrow keys** — Navigate results
4. **Press Enter** — Open the selected file

On very large workspaces you may see an honest **partial Library** notice — see [Library and very large workspaces](library-and-very-large-workspaces.md).

## Recent Files

At the top of Quick Open, you'll see **Recent Files** — the files you've accessed most recently. This makes it fast to jump back to what you were just working on.

## Filter Tabs

Quick Open shows tabs to narrow your search:

- **All** — Everything in your Library
- **Skills** — Just workflow instructions
- **Memory** — Just your memory files

## Context-Aware Icons

Each result shows an icon indicating what type of file it is:
- 📄 Regular documents
- ⚡ Skills
- 🧠 Memory files
- 📁 Folders

## Tips

- **Fuzzy matching** — You don't need exact names; "mtg prep" finds "meeting-external-prep.md"
- **Recent files first** — Files you've accessed recently appear higher in results
- **Works from anywhere** — Quick Open works whether you're in a conversation or browsing the Library
