# Keyboard Shortcuts

Use keyboard shortcuts to work more efficiently with Mindstone Rebel.

> **Tip:** View all shortcuts in **Settings → Shortcuts**.

## Quick Reference

| Shortcut | Action |
|----------|--------|
| ⌘/Ctrl+N | New chat |
| ⌘/Ctrl+O | Quick Open file (filter by All/Skills/Memory) |
| Ctrl+Tab | Next active session |
| Shift+Ctrl+Tab | Previous active session |
| ⌘/Ctrl+I | Open Actions |
| ⌘/Ctrl+Shift+N | Open Scratchpad (quick capture) |
| ⌘/Ctrl+Enter | Archive conversation (when input empty and idle) |
| Enter | Send message (queues automatically when agent is busy) |
| Enter | Stop recording (when mic is active) |
| Alt/Option+Enter | Queues your message while the agent is busy (same as Enter). To send immediately and interrupt, use the **Send now** button. |
| Ctrl+Alt+Space | Voice activation with screenshot (customizable) |
| Press voice hotkey again | Send voice memo with screenshot |
| Escape | Exit voice mode |
| Escape Escape | Stop running turn (double-tap quickly) |
| ⌘/Ctrl+F | Find in page (search within conversations) |
| ⌘/Ctrl+G | Find next match |
| ⌘/Ctrl+Shift+G | Find previous match |
| ⌘/Ctrl+↑ | Edit last message (when composer empty) |
| Ctrl+⌘+F (Mac) / F11 (Win/Linux) | Toggle fullscreen |
| ⌘/Ctrl+Plus | Zoom in |
| ⌘/Ctrl+Minus | Zoom out |
| ⌘/Ctrl+0 | Reset zoom |
| Right-click | Context menu (cut/copy/paste) in any input field |

**Platform note:** On Mac, most shortcuts use ⌘ (Command). On Windows/Linux, use Ctrl. Exception: Ctrl+Tab uses Ctrl on all platforms.

## Global Voice Hotkey

The global voice hotkey (default: **Ctrl+Alt+Space**) works even when Rebel is in the background. When you press it:

1. **Screenshot captured** — Rebel takes a screenshot of whatever you were looking at
2. **Window focuses** — Rebel comes to the foreground
3. **Voice recording starts** — Speak your message
4. **Press again to send** — Your voice message is sent along with the screenshot

This lets you ask questions about anything on your screen — a document, a website, an error message — without needing to manually take a screenshot first.

### Screenshot Privacy

Screenshots are captured directly into memory and are never saved to disk. They only become part of your conversation when you send the message. If you cancel the voice recording, the screenshot is discarded.

On Mac, you'll need to grant **Screen Recording** permission for screenshots to work. If you haven't granted this, you'll see a note in the toast saying "no screenshot" — your voice message will still work, just without the visual context.

## Customizing Shortcuts

The voice activation hotkey can be customized:

1. Open **Settings** (gear icon)
2. Go to **Agent & Voice → Voice**
3. Click **Global voice activation hotkey**
4. Press your preferred key combination
