# Restarting Onboarding or Re-running Setup Steps

There are **two different kinds of “do onboarding again”** in Rebel:

1. **Restart the full setup wizard**
2. **Re-run a specific part of setup without starting over**

They are not the same thing, and life is easier when you pick the right one.

## Restart the full onboarding wizard

If you want the full guided setup again from step 1:

1. Open **[Settings](rebel://settings)**
2. Go to **Advanced**
3. Open **Onboarding & Actions**
4. Click **Restart full onboarding**

This reopens the guided setup flow, including:

- **Welcome**
- **Spaces**
- **AI setup** using the current provider cards:
  - **ChatGPT Pro**
  - **OpenRouter**
  - **Anthropic**
- **Voice**
- **Connect your tools** using the current connector cards and search

### What this does not do

- It does **not** factory-reset Rebel
- It does **not** delete your conversations
- It does **not** wipe your current settings just to be dramatic

It simply takes you back through the guided setup so you can re-check or change things cleanly.

## Reset just the onboarding checklist

If you only want the **Getting Started checklist** to begin again:

1. Open **[Settings](rebel://settings)**
2. Go to **Advanced**
3. Open **Onboarding & Actions**
4. Click **Reset checklist**

This resets the checklist back to step 1 **without** reopening the full wizard.

Use this when the setup itself was fine, but you want the checklist prompts again.

## Revisit one setup area without restarting onboarding

Most of the time, this is what you actually want.

### AI provider setup

Go to **[Settings → Agent & Voice](rebel://settings/agents)**.

You'll see the same card-based provider setup used in onboarding:

- **ChatGPT Pro**
- **OpenRouter**
- **Anthropic**

You can connect a new provider, switch providers, or update your credentials there.

### Connectors and tools

Go to **[Settings → Connectors](rebel://settings/tools)**.

This uses the current searchable connector catalogue and connection cards. You can add, remove, or reconnect tools whenever you like — no onboarding restart required.

### Voice

Go to **[Settings → Agent & Voice](rebel://settings/agents)** to update your microphone, transcription provider, or spoken voice.

### Spaces and workspace

Go to **[Settings → Workspace](rebel://settings/spaces)** to adjust your library location or spaces.

## Re-run the discovery bits, not the wizard

If what you want is **fresh goals, updated use cases, or more context in memory**, you usually do **not** need to restart onboarding.

- Open **[The Spark](rebel://usecases)** if you want help discovering new use cases
- Ask Rebel to help you revisit goals
- Ask Rebel to refresh memory from your connected tools

That is more like a tune-up than a restart.
