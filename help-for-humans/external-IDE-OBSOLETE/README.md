# External IDE Documentation

This folder contains documentation for using Mindstone Rebel with external IDEs like Cursor and Claude Code.

**Note**: The [Rebel desktop app](../Rebel-interface.md) is the primary interface for Mindstone Rebel. These docs are for users who prefer to use Cursor or Claude Code as an alternative interface.

## Contents

- [Cursor.md](Cursor.md) - Cursor IDE setup and configuration
- [Cursor-Privacy-Policy.md](Cursor-Privacy-Policy.md) - Cursor's privacy practices and data handling

## When to use an external IDE

- You're a developer who prefers IDE-based workflows
- You need features not yet available in the Rebel app
- You want to use Cursor's codebase indexing for large projects

## Limitations vs Rebel app

- No bundled Node environment
- No app-level system prompt configuration
- No Rebel-specific automations or flows
- Manual MCP configuration required
