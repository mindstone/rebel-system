# Homepage

The **Homepage** is Rebel's landing surface — the screen you see when you open Rebel, start fresh, or come back after a break.

Its job is simple: help you decide what matters next without making you rummage around for it.

## How Homepage is laid out

Homepage combines:

- a fast **chat box** for starting something new
- recent conversation shortcuts
- the three main sections: **Today**, **Chat**, and **Coach**

## Today

**Today** is the “what needs attention now?” column.

You may see things like:

- upcoming meetings
- actions worth reviewing today
- recent automation results
- connector nudges if Rebel would be more useful with another tool connected
- Focus-related nudges if that part of Rebel is available but not yet switched on

Today refreshes while Homepage is visible, so it stays reasonably current instead of becoming decorative.

## Chat

**Chat** helps you start quickly and resume quickly.

This part of Homepage gives you:

- a prominent chat input for kicking off a new conversation
- recent conversation shortcuts so you can jump back in without hunting through history

If you already know what you want, this is the fastest route.

## Coach

**Coach** is where Rebel helps you decide what deserves attention next.

### Daily recommendations

Coach can show Rebel's take on your priorities: a short ranked set of recommendations based on your recent conversations, meetings, and goals, each with a concrete next step.

In the app, this is controlled by **Daily recommendations** in **Settings → Agent & Voice → Intelligence**:

- **Ask me each time** — Homepage shows a prompt card so you can generate recommendations on demand
- **Run automatically** — Rebel generates them for you
- **Off** — no daily recommendations

If you're using **Ask me each time**, the Coach section shows the prompt card directly on Homepage. That is the current on-demand flow.

### Coaching insights

After a meaningful conversation, Rebel may generate follow-up insights — ideas, angles, or next steps you might want to explore.

Each insight links back to the conversation it came from, so it isn't just dispensing wisdom from a mountaintop with no footnotes.

#### How coaching stays relevant

- **Insights expire after 2 days**
- Rebel prioritises the kinds of suggestions you actually act on
- Automations can also feed useful signals into Coach

### Use case cards

Coach also shows personalised use case cards — suggestions for things Rebel can help with based on your setup and how you work.

If one looks useful, click it to start with that prompt already prepared.

## What Homepage is called

The canonical user-facing name is **Homepage**.

Older docs and older conversations may still mention **The Spark**, but that is the old name for this surface.

## Customization

Homepage adapts as you use Rebel:

- **Today** changes with your meetings, actions, and automation output
- **Chat** reflects your recent conversations
- **Coach** becomes more relevant as Rebel learns which suggestions you act on

So yes, it gets better with use. Annoyingly sensible.
