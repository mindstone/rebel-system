# GitHub

Access your repositories: search code, create and manage issues, review pull requests, and run Actions.


## What You Can Do

- **Search** repositories and code
- **Issues**: Create, update, and comment on issues
- **Pull Requests**: Review, approve, merge PRs
- **Actions**: Trigger and monitor workflow runs
- **Explore** repository structure and contents


## Setup

1. Open **Settings → Connectors**
2. Find **GitHub** and click **Set up with Rebel**
3. Sign in to your GitHub account
4. Authorize Rebel to access your repositories


## Tips

- **Code search**: "Find where we handle user login in my repos"
- **PR workflow**: "Show open PRs waiting for my review"
- **Issue triage**: "List high-priority issues in the frontend repo"
