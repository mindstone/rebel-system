# Gong

Access your Gong sales intelligence data: browse call recordings, read full transcripts, search conversations, and view user activity stats.


## What You Can Do

- **List calls**: Browse recent calls with optional date filtering and pagination
- **Get call summaries**: AI-generated summaries with key points, topics, and action items
- **Get transcripts**: Read full timestamped transcripts with speaker attribution
- **List users**: See all users in your Gong workspace
- **Search calls**: Find calls by date range, host, or keyword


## Setup

1. Open **Settings → Connectors**
2. Find **Gong** and click **Set up**
3. Click **Open Gong** to go to your API settings (requires Technical Administrator role)
4. Click **Get API credentials**
5. Copy the **Access Key** and **Access Key Secret**
6. Paste both values back in Rebel

> **Plan requirement**: Gong API access requires an **Enterprise plan** with API enabled. Contact your Gong admin if API settings are not visible.


## Tips

- **Search by date**: "Show my Gong calls from last week" or "Find calls from January"
- **Get transcripts**: "Get the transcript from my call with Acme" -- Rebel fetches the full speaker-attributed transcript
- **Team activity**: "Who on the team has had the most calls this month?" -- uses user stats
- **Call review**: "Summarise the key points from yesterday's demo call" -- Rebel reads the transcript and highlights takeaways
