# Kling AI

Generate videos with AI: create videos from text prompts or transform images into video clips.


## What You Can Do

- **Text-to-video**: Generate videos from text descriptions
- **Image-to-video**: Animate still images into video
- **Check status**: Monitor video generation progress
- **Check balance**: View your API credits


## Setup

Kling requires API credentials:

1. Open **Settings → Connectors**
2. Find **Kling** and click **Set up with Rebel**
3. Enter your Kling API credentials (get them from [Kling API Keys](https://app.klingai.com/global/dev/api-key))


## Alternatives

Other video generation tools with MCP support:

- [Runway MCP](https://github.com/runwayml/runway-api-mcp-server) — Official Runway video generation
- [Luma MCP](https://github.com/lumalabs/luma-api-mcp) — Official Luma Dream Machine
- [Replicate MCP](https://replicate.com/docs/reference/mcp) — Access multiple video models
- [fal.ai MCP](https://docs.fal.ai/model-apis/mcp) — Multi-model video platform
- [MiniMax MCP](https://github.com/MiniMax-AI/MiniMax-MCP) — Official MiniMax video generation

Each platform has different strengths in video quality, style, and pricing. Explore alternatives based on your creative needs.
