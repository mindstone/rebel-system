# Brave Search

Use Brave's independent search index for web searches. Brave Search replaces Rebel's built-in DuckDuckGo-based search, providing faster, more reliable results without rate-limit interruptions.


## What You Can Do

- **Web search** — search the web using Brave's own independent index
- **Local search** — find local businesses, places, and points of interest
- **No rate limits** — your own API key means no shared usage caps


## Why Connect Brave Search?

Rebel has a built-in web search that works out of the box. However, it uses DuckDuckGo's public HTML pages, which can hit rate limits or CAPTCHAs during heavy use. When you connect Brave Search, Rebel automatically switches to it for all web searches — faster, more reliable, and no interruptions.

If you disconnect Brave Search later, Rebel falls back to the built-in search automatically.


## Setup

1. Open **Settings → Connectors** in Rebel
2. Find **Brave Search** and click **Set up**
3. Go to [brave.com/search/api](https://brave.com/search/api/) and sign up for an API key
4. Copy your API key and paste it into Rebel
5. Done — Rebel will now use Brave Search for all web lookups

> **Free tier**: Brave Search offers a free tier with 2,000 queries per month, which is plenty for most individual use.


## Tips

- You don't need to mention Brave Search by name — just ask Rebel to search for anything and it will use Brave automatically
- Works well alongside other connectors — Brave handles web search while Slack, Gmail, etc. handle their own domains
