# ClickUp

Manage your ClickUp workspace from Rebel: create and update tasks, track time, search docs, build reports, and orchestrate workflows.


## What You Can Do

- **Tasks**: Create, update, and search tasks with assignees, priorities, and due dates
- **Time tracking**: Start/stop timers and log time entries
- **Workspace**: Navigate spaces, folders, and lists
- **Docs**: Create and edit documents and pages
- **Chat**: Send messages to ClickUp channels
- **Reports**: Build status updates and summaries from your project data


## Setup

1. Open **Settings → Connectors**
2. Find **ClickUp** and click **Set up**
3. Sign in to ClickUp in the browser window that opens
4. Select the workspace to connect
5. Review permissions and click **Allow**

> **Note**: Available on all ClickUp plans. Each workspace requires a separate connection.


## Tips

- **Task management**: "Create a task in the Sprint 4 list assigned to me, due Friday"
- **Status updates**: "What tasks are in progress in the Engineering space?"
- **Time tracking**: "Start tracking time on the code review task" or "Log 2 hours on yesterday's research"
- **Reports**: "Build a summary of completed tasks this week"
- **No deletion**: ClickUp currently disables deletion via MCP as a safety measure — use the ClickUp app directly for that
- **Rate limits**: If Rebel slows down during heavy ClickUp work, wait a moment — ClickUp's API rate limits apply
