# Smartsheet

Access your Smartsheet workspace: list sheets, read rows, create and update data, and search across sheets.


## What You Can Do

- **List sheets** — browse all sheets in your workspace
- **Read rows** — get data from specific sheets
- **Create and update rows** — add or modify sheet data
- **Search** — find content across your Smartsheet workspace


## Setup

1. Open **Settings → Connectors** in Rebel
2. Find **Smartsheet** and click **Set up**
3. Go to your Smartsheet account → **Account → Apps & Integrations → API Access** to generate a token
4. Copy the token and paste it into Rebel
5. Select your **region** (US, EU, or AU) if you're not on the default US region

> **Permissions**: The API token inherits the permissions of the user who creates it. Use an account with appropriate access to the sheets you want Rebel to work with.


## Tips

- "List my Smartsheet sheets" to see what's available
- "Find rows in [sheet name] where status is overdue" for filtered queries
- "Add a row to [sheet name] with these values" for quick data entry
