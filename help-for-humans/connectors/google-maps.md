# Google Maps

Get directions, find places, convert addresses to coordinates, and calculate distances between locations.


## What You Can Do

- **Get directions** between two locations with route details
- **Search places** by name or category near a location
- **Geocode addresses** to coordinates (and reverse)
- **Calculate distances** and travel times between multiple points
- **Look up elevation** data for any location
- **Get place details** like ratings, hours, and contact info


## Setup

Google Maps requires an API key from Google Cloud:

1. Open **Settings → Connectors**
2. Find **Google Maps** and click **Set up with Rebel**
3. Go to [Google Cloud Console](https://console.cloud.google.com/apis/credentials) and create an API key
4. Enable these APIs: Geocoding, Places, Directions, Distance Matrix, Elevation
5. Paste your API key in Rebel

New Google Cloud accounts get $200/month in free credits. See [pricing details](https://developers.google.com/maps/billing-and-pricing).


## Example Prompts

- "How far is it from our office to the airport?"
- "Find Italian restaurants within 1km of 350 5th Ave, New York"
- "What are the coordinates for 1600 Amphitheatre Parkway?"
- "Plan a route from San Francisco to Los Angeles with stops in San Jose and Monterey"
