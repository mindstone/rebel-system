# Fathom

Access your Fathom meeting recordings: list meetings, read transcripts, get AI summaries and action items.


## What You Can Do

- **List meetings** from your Fathom account
- **Read transcripts** with speaker identification
- **Get summaries** and action items from recorded calls
- **Browse teams** and team members


## Setup

Fathom requires an API key:

1. Open **Settings → Connectors**
2. Find **Fathom** and click **Set up with Rebel**
3. Enter your Fathom API key (get it from [Fathom API Settings](https://fathom.video/customize#api-access-header))


## Alternatives

Other meeting transcript tools with MCP support:

- [Fireflies](https://docs.fireflies.ai/getting-started/mcp-configuration) — Official MCP server with OAuth
- [Otter](https://help.otter.ai/hc/en-us/articles/35287607569687-Otter-MCP-Server) — Official MCP server
- [Grain](https://grain.com/release-note/06-18-2025) — Official MCP endpoint
- [MeetGeek](https://github.com/meetgeekai/meetgeek-mcp-server) — Open-source MCP server
- [tl;dv](https://github.com/tldv-public/tldv-mcp-server) — Open-source MCP server

Rebel's Fathom integration uses the [Fathom API](https://help.fathom.video/en/articles/8368641-public-api) directly. Choose the tool that best fits your meeting recording workflow.
